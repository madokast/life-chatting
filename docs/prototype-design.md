# 静态原型设计

本项目采用 HTML+CSS 实现静态原型，用于定义应用的页面结构和样式，为后续开发提供基础。

## 设计要求

### 响应式设计
- 页面布局需支持不同屏幕尺寸，包括手机、平板和桌面设备
- 使用 CSS 媒体查询或弹性布局等技术实现响应式设计
- 确保在各种设备上都有良好的用户体验

### 图片资源
原型中使用的图片统一使用以下 URL 格式：
```html
<img src="https://cdn.pixabay.com/photo/2016/11/14/16/20/snowflake-1823942_640.jpg" alt="test-image">
```

### 页面结构
根据 README 中定义的页面结构，原型应包含：
- 底部导航栏（Bottom Navigation/Tab Bar）
  - 日记页面
  - 聊天页面
- 左侧抽屉菜单（Left-side Navigation Drawer）
  - 配置
  - 同步
  - 导出
  - 导入

## 页面设计

### 日记页面

#### 单条日记结构
- **日期时间信息**：显示格式为 `2025-12-31 14:15:16`，位于日记顶部左侧
- **操作菜单**：右侧有 ⋮ 按钮，点击后弹出以下选项：
  - 删除
  - 修改
  - 聊天
  - 折叠
- **日记内容**：
  - 内容超过 3 行会自动折叠
  - 点击日记内容可展开全部内容

#### 图片处理
- **1 个图片**：
  - 显示在日记内容右侧
  - 占据 3 行高度
  - 图片容器为正方形
  - 非正方形图片会缩放后放入方形容器中

- **2 个及以上图片**：
  - 显示在日记内容下方
  - 水平铺开显示
  - 自动检测一行最多显示的图片数目
  - 超出部分显示 … 表示有图片折叠，不换行
  - 点击 … 处展开日记内容，所有图片水平铺开，超出自然换行

#### 图片全屏预览
- 点击图片可全屏显示
- 全屏显示时右上角有 × 按钮，点击关闭全屏预览

### 聊天页面

#### 聊天界面结构
- **顶部标题栏**：显示聊天主题或 AI 名称
- **消息列表**：展示用户和 AI 的对话历史
  - 用户消息：靠右显示，带有用户头像
  - AI 消息：靠左显示，带有 AI 头像
- **输入区域**：
  - 文本输入框：支持多行输入
  - 发送按钮：发送消息
  - 附件按钮：支持添加图片等附件
  - 语音输入按钮：支持语音转文字

#### 消息样式
- **用户消息**：
  - 背景色：蓝色系
  - 文字颜色：白色
  - 圆角设计：右上角圆角较小，左下角圆角较大
  - 带有时间戳

- **AI 消息**：
  - 背景色：浅灰色
  - 文字颜色：黑色
  - 圆角设计：左上角圆角较小，右下角圆角较大
  - 带有时间戳

#### 交互功能
- **消息发送**：点击发送按钮或按 Enter 键发送消息
- **消息滚动**：新消息自动滚动到视图底部
- **输入框自动高度**：根据输入内容自动调整高度
- **AI 思考状态**：显示 AI 正在思考的动画效果

### 响应式布局
- 适配不同屏幕尺寸
- 在移动端和桌面端均有良好的显示效果

## 技术实现

- 使用纯 HTML+CSS 实现，不依赖 JavaScript
- 采用现代化的 CSS 技术，如 Flexbox、Grid 等
- 确保代码结构清晰，便于维护和扩展

## 开发流程

1. 创建 HTML 结构文件
2. 添加 CSS 样式，实现响应式设计
3. 测试不同设备上的显示效果
4. 调整优化，确保用户体验良好
